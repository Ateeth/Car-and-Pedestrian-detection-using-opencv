Car Tracker

Using the car tracker functionality of this webapp you can track cars in an image or video.
Most of the scanned cars in the image or video will be displayed in
will be displayed in rectangular boxes
Car detection will help in self driving and can help
in notifying the driver if there is a car ahead or not and can hence be helpful in preventing accidents
Below there are 2 card options that are provided.
The options provided are
  • Car tracking using an existing image
  • Car tracking using an existing video

IMAGE CAR TRACKER

 Click this card if you want to use a script which highlights the cars in the image selected.

VIDEO CAR TRACKER

 Click this card if you want to use a script which highlights the cars in the video selected.
