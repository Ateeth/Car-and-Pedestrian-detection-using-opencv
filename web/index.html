Improving Self-Driving functionality

Using this web app on the existing images or videos provided in this app
Cars and Pedestrians can be scanned
which can help improve the functionality of pedestrian protection in Self Driving
and Car detection as well in Self Driving to inform if there is a car ahead or not
which helps prevent accidents
Below there are 4 card options that are provided.
The options provided are
  • Car tracker
  • Pedestrian tracker
  • Both Car and Pedestrian tracker
  • Car Speed tracker

CAR TRACKER

 Click this card if you want to use a script which highlights the cars in the image or video selected to some extent.

PEDESTRIAN TRACKER

 Click this card if you want to use a script which highlights the pedestrians in the image or video selected to some extent.

CAR AND PEDESTRIAN TRACKER

 Click this card if you want to use a script which highlights both cars and pedestrians in the image or video selected to some extent.

CAR SPEED TRACKER

 Click this card if you want to use a script which shows the speed of a few cars in the video selected to some extent.

VIDEO CAR COUNTER

 Click this card if you want to use a script which highlights the cars in the video selected and also will try to display the number of cars in the video.
